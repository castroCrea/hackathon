#Overview


## Create a game

POST `/api/games`

###Body 

    {
        "title": "space invador",
        "description": "in the space",
        "maxPlayer": 0
    }
    
## Create Player

###First the game master

POST `api/players`

####Body

    {
      "name": "Regis",
      "picture": "json_encode(md6)",
      "gameMaster": "/api/games/1",
      "race": "/api/races/1",
      "job": "/api/jobs/1",
      "gender": "/api/genders/1"
    }

###First the player

POST `api/players`

####Body

    {
      "name": "Regis",
      "picture": "json_encode(md6)",
      "game": "/api/games/1",
      "race": "/api/races/1",
      "job": "/api/jobs/1",
      "gender": "/api/genders/1"
    }
    
##Start Game

Only game master

###header

Authorization -> user_token

PUT `/api/games/start/{game_id}`
    
##End Game

Only game master

Authorization -> user_token

PUT `/api/games/end/{game_id}`
    
##Finish Game

Only game master

Authorization -> user_token

PUT `/api/games/finish/{game_id}`
